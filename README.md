# 第二章数据可视化网页

## 项目介绍

本项目是基于第二章Jupyter笔记本内容创建的交互式数据可视化网页，展示了10种不同类型的图表，采用浅紫色主题设计，具有响应式布局和丰富的交互功能。

## 功能特点

### 📊 10种图表类型
1. **基础折线图** - 展示多组数据线的趋势变化
2. **温度折线图** - 未来15天最高最低温度趋势分析
3. **基础柱状图** - 单组数据柱状图展示
4. **堆积柱状图** - 多组数据堆积展示
5. **饼图** - 数据占比分布可视化
6. **散点图** - 变量相关性分析
7. **面积图** - 数据累积展示效果
8. **气泡图** - 三维数据可视化
9. **雷达图** - 多维度数据对比
10. **热力图** - 数据密度分布展示

### 🎨 交互功能
- **悬停查看详情** - 鼠标悬停显示具体数据值
- **动态缩放** - 支持图表区域的缩放操作
- **数据筛选** - 可控制显示/隐藏数据系列
- **实时更新** - 点击按钮可生成新的随机数据
- **响应式设计** - 适配不同屏幕尺寸的设备

### 🎯 技术实现
- **HTML5** - 语义化结构布局
- **CSS3** - 现代响应式设计，浅紫色主题
- **JavaScript** - 动态交互逻辑
- **Plotly.js** - 专业的数据可视化库
- **Flexbox/Grid** - 灵活的布局系统

## 使用说明

### 网页访问
直接打开 `chapter2_index.html` 文件即可在浏览器中查看网页。

### 导航功能
- 顶部导航栏可快速切换不同图表类型
- 首页提供所有图表的预览卡片，点击卡片可直接跳转

### 图表操作
- **基础折线图**：点击"刷新数据"按钮生成新数据
- **温度折线图**：通过复选框控制最高/最低温度显示
- **柱状图系列**：点击按钮更新数据
- **饼图/散点图等**：支持随机数据生成

### 响应式特性
- 桌面端：完整导航栏和图表展示
- 平板端：自适应布局调整
- 手机端：垂直导航和紧凑布局
